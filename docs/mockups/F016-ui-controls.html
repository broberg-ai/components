@broberg/ui-controls-coreF016 · mockup

Suiten der erstatter hver native control huset ikke kan udstå — CustomSelect, DatePicker, ConfirmModal og Toast. Ét headless-core (kalender-matematik, tastatur-FSM, toast-kø) driver React- og Preact-adaptere. Alt herunder er ægte og klikbart.

CustomSelect

Portal-listbox i stedet for <select> — tastatur-navigerbar (↑↓ · ↵ · esc), afkryds valgt.

DatePicker

Mandag-først dansk kalender · format DD-MM-ÅÅÅÅ · Today/Ryd. Ingen native type=date.

ConfirmModal

Scrim + ESC + tone (fare/primær) i stedet for window.confirm(). Bekræft viser loading.

Toast

Stak øverst-højre, auto-dismiss med progress, manuel luk, kappet ved 3. Erstatter alert().
